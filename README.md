## Necessary Links
**/ -> Home Page**                                 
**register/ -> Register Page**                            
**profile/ -> Profile Page**                              
**profile/update/ -> Profile Update Page**                                                  
**login/ -> Login Page**                                    
**logout/ -> Logout Page**                                              
**add_tweet/ -> to Add a Tweet**                                            
**hashtag/hashtag-name/ -> For The HashTag**                                                  
**quests/ -> Returns All Tweets**                              
**search/<> -> Return a Search**
